HF-WS2812 is an ESP32-specific driver for WS2812 and compatible addressable RGB(W) LED strips (commonly known as NeoPixels). The driver uses the ESP32â€™s RMT (Remote Control) peripheral to generate the precise timing signals required by WS2812 LEDs, providing hardware-accelerated LED control with minimal CPU overhead.

The driver supports both C and C++ APIs, includes built-in animation effects, and is fully configurable via ESP-IDFâ€™s Kconfig system. Itâ€™s compatible with WS2812, WS2812B, WS2813, SK6812, and other WS2812-compatible LED variants.

// Create strip: GPIO, RMT channel, LED count, LED type

void app_main(void) {
    strip.begin();
    anim.setEffect(WS2812Animator::Effect::Rainbow);
    
    while (true) {
        anim.tick();
        vTaskDelay(pdMS_TO_TICKS(50));
    }
}
