Datasheet Directory

This directory is intended for WS2812 LED datasheets.

Note

WS2812 is a generic LED driver IC used in addressable RGB LED strips (commonly known as NeoPixels). The driver is compatible with WS2812, WS2812B, WS2813, and similar variants.

Finding Datasheets

Datasheets for WS2812 and compatible LEDs can be found from trusted sources:

  • WS2812 Datasheet (SparkFun): https://cdn.sparkfun.com/datasheets/Components/LED/WS2812.pdf
  • Adafruit NeoPixel Guide: https://learn.adafruit.com/adafruit-neopixel-uberguide

Driver Compatibility

The driver supports:

  • WS2812: Original version
  • WS2812B: Improved version with better timing
  • WS2813: Version with backup data line
  • SK6812: Compatible variant
  • Other WS2812-compatible LEDs

Timing Configuration

The driver allows runtime configuration of timing parameters to support different LED variants and ensure accurate signal generation.
